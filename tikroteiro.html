Conectar Supabase

O Supabase é gratuito e vai salvar todos os seus roteiros, vídeos e hashtags na nuvem — acessíveis de qualquer dispositivo.

1
Acesse supabase.com e crie uma conta gratuita
2
Crie um novo projeto (dê um nome e senha)
3
Vá em Settings → API e copie a Project URL e anon public key
4
Vá em SQL Editor, cole o SQL abaixo e clique em Run para criar as tabelas
CREATE TABLE roteiros ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, produto text, conteudo text, tipo text, created_at timestamptz DEFAULT now() ); CREATE TABLE videos ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, produto text, prompt text, plataforma text, estilo text, created_at timestamptz DEFAULT now() ); CREATE TABLE hashtags ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, produto text, tags jsonb, created_at timestamptz DEFAULT now() );
